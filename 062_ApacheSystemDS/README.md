# about
Apache SystemDS (formerly SystemML) describes statistics / ML pipelines declaratively
and executes them on an embedded Java runtime (no external database). Requires Java 11+.

# python
```bash
python3.12 -m venv env
source env/bin/activate
pip install --upgrade pip setuptools wheel
pip install systemds numpy
```

```bash
python sample1.py
```

# Deactivate Virtual Environment
```bash
(env) $ deactivate
```
