# prepare
```
mkdir -p ./files
```

# build and run
```
docker compose up -d
```
Access the following URL using the Web browser.
* apache/hop
    http://localhost:8083/
    account  : cluster
    password : cluster

* apache/hop-web
    http://localhost:8082/ui

# down
```
docker compose down
```
